virtual bool LevelStorageSource::isNewLevelIdAcceptable ( const std::string &  levelId)
pure virtual

Tests if a levelId can be used to store a level. For example, a levelId can't be called COM1 on Windows systems, because that is a reserved file handle.

Also, a new levelId may not overwrite an existing one.
